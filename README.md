# 回膛：磁弹猎人

公开试玩构建：`local-20260719-audio-reliability-6`

这是一个三分钟 Graybox 原型，用于验证移动、射击、回收和磁场组合的基础战斗循环。

## 本次音频可靠性迭代

- 五首长音乐改为 160 kbps MP3，减少移动网络等待。
- 射击、回膛命中、开始、武器选择的 15 个关键短音效被打入单一脚本包，不再依赖游戏过程中临时发起多个 WAV Range 请求。
- 音乐播放失败自动重试，不再静默吞掉错误。
- 支持暂停、恢复和页面前后台切换后的音乐恢复。
- 新增冷启动、故障注入与多轮音频状态测试。

## 手机测试说明

- 游戏按横屏操作设计。
- 部分 Android 微信内核不开放网页方向锁定接口；必要时请横置手机，或从微信菜单选择“在浏览器打开”。
- GitHub Pages 托管不会绕过微信内核自身的方向锁限制。

当前仍是 `Graybox prototype`，Gate 2 保持 fail，等待真实 Android 微信完成横屏操作和实际听感复验。
