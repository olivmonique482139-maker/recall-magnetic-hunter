# 回膛：磁弹猎人

公开试玩构建：`local-20260719-mobile-entry-2`

这是一个三分钟 Graybox 原型，用于验证移动、射击、回收和磁场组合的基础战斗循环。

## 手机测试说明

- 游戏只按横屏操作设计。
- 从微信打开时，当前部分 Android 微信内核无法由网页自动锁定横屏。
- 如果微信没有随手机旋转，请通过右上角菜单选择“在浏览器打开”，然后横置手机。
- 该限制来自微信网页容器，不会因为 GitHub Pages 托管而消失。

Gate 2：fail，等待真实手机横屏操作复验。
