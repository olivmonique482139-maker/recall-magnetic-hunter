# 回膛：磁弹猎人

公开试玩构建：`local-20260719-mobile-pause-audio-9`

这是一个三分钟 Graybox 原型，用于验证移动、射击、回收和磁场组合的基础战斗循环。

## 本次手机暂停与首响迭代

- 暂停改为全屏独立界面，战斗计时、输入、音乐和音效全部冻结。
- 暂停内容支持物理横屏和微信竖持旋转坐标下的手指滑动。
- “继续／重来／退出”同一行、等宽、同字号；重来保留武器并清空本局状态。
- 进入战斗前验证战斗音乐已经实际播放推进，不再先静音进入。
- 微信拒绝启动音频时显示可点击的重试界面。
- 关键音效和选角／战斗音乐继续使用内嵌音频包，减少移动网络请求。

## 手机测试说明

- 游戏按横屏操作设计。
- 腾讯 CloudBase 链接是国内微信测试的推荐地址；GitHub Pages 仅作为公开备份。
- 部分 Android 微信内核不开放网页方向锁定接口，必要时请横置手机。

当前仍是 `Graybox prototype`，Gate 2 保持 fail，等待真实 Android 微信完成实际听感与手指操作复验。
